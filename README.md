# DinoCPP

DinoCPP is not a game, it is a very bad C++ code(

This game write on a C++ with [sfml](https://www.sfml-dev.org/index.php) library.

### Install sfml

Debian/Ubuntu
```sh
apt-get install libsfml-dev
```
Arch Linux
```sh
pacman -S sfml
```
### Build
On Linux:
```sh
g++ main.cpp -lsfml-graphics -lsfml-window -lsfml-system
```
